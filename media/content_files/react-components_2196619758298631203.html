React has 2 different ways of creating components

  • Function Components
  • Class components

Function Components

The simplest way to define a component is js function:


function Welcome() {
return 
Hello, World
;
}
  • We call such components “function components” because they are literally JavaScript functions.
  • Components start with capital case.

2 - Class Components

These two components are equal


class Welcome extends React.Component {
render() {
  return 
Hello, World
;
}
}
